## About The Project

This repository contains the source code for the DevThemes Digital Storefront, a curated collection of high-performance web templates and components tailored for developers. It is built with a modern, mobile-first architecture and serves as an autonomous, data-driven platform that seamlessly integrates with my main portfolio ecosystem.

It is designed with a premium, glassmorphic UI, fluid micro-animations, and dynamic visual layouts that scale flawlessly from ultra-small mobile screens to massive desktop displays.

**Live View:** [https://devthemes-store.vercel.app](https://devthemes-store.vercel.app)

## Key Features

- **Blazing Fast Performance:** Powered by the Next.js App Router, generating pure static HTML for instant page loads.
- **Flawless Responsiveness:** Mathematically perfect layouts down to 320px mobile screens, powered by Tailwind CSS v4.
- **Enterprise SEO:** Out-of-the-box OpenGraph generation, programmatic dynamic sitemaps, and strict robots.txt routing for lightning-fast indexing.
- **Glassmorphic Design System:** Complex, high-contrast blurred UI components, gradient orbs, and CSS micro-animations.
- **Data-Driven Architecture:** Entirely powered by a lightweight, Zod-validated static data layer preventing runtime exceptions.

## Technical Stack

### Frontend Architecture
- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Validation:** [Zod](https://zod.dev/)

### Integrations
- **Deployment & Hosting:** Vercel

## Local Development

Want to run this project locally? Follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.
